# Kubo's Shop 購物網站
![image](https://github.com/zz860619/shopping-demo/blob/master/demo/1576632958584.jpg)
## Demo
<https://kubos-shop.herokuapp.com/>
## 介紹
主要為個人練習所建立的購物網站。

**前端**

**前台:**
* 首頁banner、商品輪播
* 購物車功能(使用localStorage)
* 商品列表、依商品分類展示
* 單一商品詳細內容介紹
* 串接Google帳戶登入
* 結帳資訊
* 表單驗證

**後台**
* 商品管理(新增、刪除、編輯)

**後端**
* 架設資料庫
* API路由設計

## 主要技術

* React
* Redux
* React-router
* axios 呼叫後端Api
* Bootstrap4 + 自定義CSS
* LocalStorage
* Formik表單 + yup驗證規則工具
* Express
* MongoDB
* Passport
